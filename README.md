# Restaurant QR Menu System - User Guide

## How to Use

### Admin Access
1. Go to: https://qr-menu-system-5ked.onrender.com/admin
2. Password: [give them the password]

### Adding Menu Items
1. Click "Menu Manager"
2. Fill in item details
3. Upload photo
4. Click "Add to Menu"

### Managing Orders
1. Go to "Orders Dashboard"
2. See new orders in real-time
3. Click "Mark as Complete" when done

### Creating Discount Codes
1. Go to "Discounts"
2. Create code (e.g., "WELCOME10")
3. Set discount percentage or amount
4. Set expiration date

### Table Bookings
1. Customers go to /booking
2. Select date, time, party size
3. You receive notification
4. Confirm in Admin → Bookings

## Troubleshooting
- If orders not showing: Refresh page
- If images not loading: Check uploads folder
- For any issues: Restart the server
